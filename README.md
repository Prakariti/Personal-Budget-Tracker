# Personal Budget Tracker

## Overview
The Personal Budget Tracker is a Python-based application designed to help users manage their income, expenses, and savings efficiently. The system provides a user-friendly graphical interface and visual insights to support better financial planning.

## Objectives
- Track monthly income and expenses  
- Categorize transactions  
- Analyze spending patterns  
- Monitor savings over time  

## Features
- Add and manage income and expense records  
- Monthly transaction summary  
- Income vs expense comparison  
- Savings tracking with visual charts  
- User-friendly graphical interface  

## Technologies Used
- Python  
- Tkinter (GUI)  
- Pandas (data handling)  
- Matplotlib (data visualization)  
- CSV files (data storage)  

## Project Workflow
1. Data input through GUI  
2. Data storage in CSV files  
3. Data processing using Pandas  
4. Visualization using charts and graphs  

## Visualizations
- Income vs Expense bar chart  
- Monthly expense distribution (pie chart)  
- Savings trend over time  

## How to Run the Project
1. Clone the repository  
2. Install required libraries  
3. Run the main Python file  
   ```bash
   python main.py
